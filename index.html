The Random DevTools Project

The Random DevTools Project is an ambitious project that I launched in the summer of 2019 to build developer tools for modern application development. The core idea of the project is to build DevTools as I need them (hence, the use of the adjective) and share them with developers around the world.

- quickbuild: a Node.js package that generates boilerplate code for a large range of project types, like Express.js server, React-Redux application, Node.js server, etc., and has added functionality, like initializing version-control, adding package-management, installing dependencies, etc., for each project.

-  Search GitHub: a Chrome extension that allows a user to construct powerful queries to search GitHub repositories, users, marketplace, etc. using 24 different parameters.

-  acquire-module: Node.js' require function on steroids.

Using the same idea, I have developed- and am developing- other DevTools; hoping that they would help serve the developer community in the same way that helped me.

A Billing and Invoicing desktop application

I worked in a team of six student software engineers to build an invoicing desktop application for the uniform shop at the British School, New Delhi. The application was written in Java (using JavaFX) and SQL, and some of its feautures included:

  • - Generating invoices
  • - Recording and keeping track of inventory
  • - Recording and keeping track of orders
  • - Notifying the student by email when an order was ready to be collected
  • - Linking data with the school's database of student accounts

Unfortunately, the school didn't open-source the project

A Tetris autoplyer AI

I redesigned the classic retro Tetris game- changing its GUI and some of its gameplay- and wrote a Genetic Algorithm for an auto-player that when finished had an average score of over 2 million and an average game-time of 92 minutes.
